HTML/CSS Basics

Défi 1 sémantique & imbrication

Règles de cet atelier : construire et concevoir un élément web uniquement avec une structure HTML sémantique (pas de classe, pas d'identifiant en CSS).

Initialisation

  • Dans un nouveau dossier , créer le fichier index.html
  • Ouvrez le fichier avec votre IDE préféré et ajoutez les balises de base HTML5 (DOCTYPE, html, head, body).
  • Créez un fichier style.css.
  • Dans le "head", ajoutez d'abord quelques balises comme "meta" ou "title" :
  • Ajoutez ensuite une balise "link" avec le lien vers style.css.
  • Dans index.html, dans le "body", ajoutez une balise "h1" avec le texte « Hello world !!! ».
  • Dans style.css, ajoutez le code :

    h1 {
    color: #a306b6;
    }
  • Ouvrez index.html dans votre navigateur et vérifiez que vous pouvez lire le message : « Hello world !!! » en orange.

    Ajouter une couleur est un bon moyen de savoir si tu as bien relié ton fichier css à ton html
